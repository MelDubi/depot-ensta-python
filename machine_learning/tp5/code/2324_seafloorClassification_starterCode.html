Rgression Logistique:
model = LogisticRegression(penalty='l2', C=1.0, solver='lbfgs', max_iter=100, class_weight=None)
  • penalty: La norme utilise pour la pnalisation ('l1', 'l2', 'elasticnet', 'none').
  • C: Inverse de la force de rgularisation; valeurs plus faibles spcifient une rgularisation plus forte.
  • solver: L'algorithme utilis dans l'optimisation ('newton-cg', 'lbfgs', 'liblinear', 'sag', 'saga').
  • max_iter: Le nombre maximum d'itrations pour l'optimisation.
  • class_weight: Spcifie le poids relatif des classes pour l'entranement dsquilibr.
Rseaux de Neurones (Multi-layer Perceptron):
model = MLPClassifier(hidden_layer_sizes=(100,), activation='relu', solver='adam', alpha=0.0001, 
                      learning_rate='constant', learning_rate_init=0.001, max_iter=200)
  • hidden_layer_sizes: La taille des couches caches.
  • activation: La fonction d'activation utilise ('identity', 'logistic', 'tanh', 'relu').
  • solver: L'optimiseur pour la minimisation de la perte ('lbfgs', 'sgd', 'adam').
  • alpha: Terme de rgularisation pour les poids.
  • learning_rate: Taux d'apprentissage pour les mises jour de poids ('constant', 'invscaling', 'adaptive').
  • learning_rate_init: Taux d'apprentissage initial utilis. C'est la taille de pas dans la mise jour des poids.
  • max_iter: Le nombre maximum d'itrations pour l'optimisation.
Random Forests:
model = RandomForestClassifier(n_estimators=100, criterion='gini', max_depth=None, 
                               min_samples_split=2, min_samples_leaf=1)
  • n_estimators: Le nombre d'arbres dans la fort.
  • max_features: Le nombre de caractristiques considrer pour la meilleure division chaque nud.
  • max_depth: La profondeur maximale de chaque arbre dans la fort.
  • min_samples_split: Le nombre minimum d'chantillons requis pour diviser un nud interne.
  • min_samples_leaf: Le nombre minimum d'chantillons requis pour tre un nud feuille.
Support Vector Machines (SVM):
model = SVC(C=1.0, kernel='rbf', degree=3, gamma='scale', class_weight=None)
  • C: Paramtre de rgularisation pour contrler le cot des erreurs.
  • kernel: Le noyau utilis pour la transformation de l'espace ('linear', 'poly', 'rbf', 'sigmoid').
  • degree: Degr du noyau polynomial (utilis avec le noyau polynomial).
  • gamma: Coefficient du noyau ('scale', 'auto' ou un nombre flottant).
  • class_weight: Spcifie le poids relatif des classes pour l'entranement dsquilibr.
