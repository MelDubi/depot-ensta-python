3.3 - Prtraitement du dataset avec tf.data.Dataset API

L'API tf.data.Dataset permet de dfinir assez simplement des chanes de traitements efficaces efficient. Ainsi, le schma classique est :

  • la cration d'un dataset source partir des donnes d'entre (dj fait ici);
  • la dfinition et l'application de transformations du datasets pour prtraiter les donnes;
  • l'itration sur le dataset et traiter tous les lments (consume the dataset).
